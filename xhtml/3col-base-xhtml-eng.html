Main content area (Base)

Normally contains the main content of the page and the date the content was last modified.

Important default values

Note: Default values are for the base theme and not the current theme.

cn-centre-col:
  • background: #FFF
cn-centre-col-inner:
  • margin-right: 25% (must equal the cn-right-col width)
  • margin-left: 25% (must equal the cn-left-col width)
  • padding-right: 10px
  • padding-left: 10px
cn-centre-col-gap (gap between the column and the footer):
  • background: #FFF

Primary navigation (Base)

Normally contains a search field and navigation links

Important default values

Note: Default values are for the base theme and not the current theme.

cn-left-col:
  • width: 25%
  • margin-right: -25% (must equal the negative of the cn-left-col width)
  • background: #FFF
cn-left-col-inner:
  • margin: 0
  • padding: 0
cn-left-col-gap (gap between the column and the footer):
  • width: 25% (must equal the cn-left-col width)
  • background: #FFF
